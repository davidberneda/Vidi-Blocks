# Vidi-Blocks
Classic blocks game made with Embarcadero RAD Studio. Pure Delphi and Firemonkey. 

### For Windows, Mac, Android, iOS and Linux

- No ads
- No fancy effects, no delays, click and play
- Full source code
  
### Steema libraries for Delphi and C++ developers

- [TeeChart](https://github.com/Steema/TeeChart-VCL-FMX-Samples)  Charting components
- [TeeGrid](https://github.com/Steema/TeeGrid-VCL-FMX-Samples)  100% Grid pure Pascal
- [TeeBI](https://github.com/Steema/TeeBI)  Data-mining, fast queries, big data, visualization
- [TeeTree](https://github.com/Steema/TeeTree) TreeView replacement control, flow-charting, diagramming
